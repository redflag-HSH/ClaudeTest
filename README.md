# ClaudeTest

Claude Code와 함께 개발 중인 Unity 2D 액션 RPG 프로토타입입니다.

## 개요

- **엔진**: Unity 6000.0.62f1 (URP 2D)
- 퀘스트, 대화, 스킬, 인벤토리, 컷신(Timeline) 등 RPG 핵심 시스템을 실험적으로 구현한 프로젝트입니다.

## 주요 구성

- `Assets/00.Scripts` — Core, Player, Enemy, Entity, Quest, Dialog, Inventory, Interaction, UI 등 게임플레이 스크립트
- `Assets/04.Dialogs` / `05.Quests` / `06.Skills` — 대화 · 퀘스트 · 스킬 데이터
- `Assets/07.TimeLine` — 컷신 연출
- `Assets/03.MaterialsNShaders` — 머티리얼 및 셰이더
